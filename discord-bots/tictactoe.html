About the Bot

Ever been bored in discord? Well why not fight the boredom with a game of Tic-Tac-Toe with one of your friends? Friends not good enough? Then challenge soem random server member to a game of pure wits!

Commands

Command Name Command Description
t!help Show a list of commands
t!game {member} Start a game with a member
t!move {board space} Attempt to make a move
t!end End the current game you are in
t!board Show your current game's board state
t!say {something} Have the bot repeat what you input
t!how Show a basic how to play of the bot
t!invite Get an invite link for the bot
t!uptime View the bot's current runtime
